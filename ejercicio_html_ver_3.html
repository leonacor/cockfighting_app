Ejemplo de formulario en HTML y Javascript

Para el siguiente ejemplo se uso el archivo html_ver_31. también se muestra la función para crear el objeto json que se descargará. se modificó la posición de la variable local let json_array y se re-declaró en varible global var json_array. cada que se presioné el botón "Guardar" se almacenará un nuevo objeto json en una nueva posición del json_array y no sobre-escriba el anterior (como lo hace el archivo archivo html_ver_31). al globalizar la variable json_array, cuando se presiona el botón, el objeto obj_json se guarda en la siguiente posición del array. para cuando el código js llamé a la función JSON.stringify(json_array) se muestren todo el contenido de la variable json_array, y no solo el json_obj actual.
